Exemplo com tooltip HTML

No exemplo simples anterior, a tooltip apresentava apenas texto (o conteúdo do atributo "title"). Porém, é possível a tooltip apresentar HTML, com imagens, etc...

Para isso:

  1. Crias uma div escondida (display:none)
  2. Dentro desta div, crias uma (ou mais) div, com o conteúdo que queres que apareça na(s) tooltip(s), atribuindo-lhe ids distintos
  3. No sítio onde queres que a tooltip apareça, para além de definires a class (que é utilizada como ponto de acesso, se bem te lembras, para a função tooltipster), adicionas um atributo extra, chamado "data-tooltip-content", e dás lhe como valor "#{idQueDesteÀDivDoConteúdo".

Exemplo
HTML:


<div class="tooltip clickable" data-tooltip-content="#conteudoDaTooltip" title="Sara Umbelina Macedo Guimarães" alt="Sara Umbelina Macedo Guimarães" >Sara Guimarães</div>

<div class="esconder" >
	<div id="conteudoDaTooltip" class="esconder" >
		<img src="img/sguimaraes.jpg" /> <span class="sguimaraes" >Sara Umbelina Macedo Guimarães</span>
	</div>
</div>

JAVASCRIPT: 


<script>
  $(document).ready(function() {
    $(".tooltip").tooltipster(
		{
			theme: "tooltipster-light"
		}
    );
});
</script>

RESULTADO:

Sara Guimarães

Sara Umbelina Macedo Guimarães
