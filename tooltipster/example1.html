Exemplo simples

Para usar o Tooltipster, precisas de adicionar 3 ficheiros ao teu HTML:

  • JQuery
  • Javascript do Tooltipster
  • CSS do Tooltipster

No caso do Weebox, o JQuery já vem carregado, e apenas precisas de adicionar os ficheiros do Tooltipster.

Numa página "limpa" (o Weebox é um caso especial), terias algo como: 

<head>
<link rel="stylesheet" type="text/css" href="css/tooltipster.bundle.min.css" />
<script type="text/javascript" src="js/jquery-2.2.4.min.js"></script>
<script type="text/javascript" src="js/tooltipster.bundle.min.js"></script>
</head>

Depois, par ativar o plugin, tens de fazer 2 coisas:

  • Adicionar uma classe de "âncora" (um ponto de acesso) nos sítios onde quiseres adicionar a tooltip especial
  • Chamar a função tooltipster() nos elementos que correspondem àquela classe

Por exemplo:

HTML:


<img class="tooltip" src="img/sguimaraes.jpg" title="Sara Guimarães" alt="Sara Guimarães" />

JAVASCRIPT: 


<script>
  $(document).ready(function() {
    $(".tooltip").tooltipster();
});
</script>

RESULTADO:

O Tooltipster disponibiliza alguns "temas" que permitem mudar o aspeto das tooltips:

  • Default
  • Light
  • Borderless
  • Punk
  • Noir
  • Shadow
Para se utilizar um tema, basta adicionar o CSS deste tema e ao executar o comando tooltipster(), dizer-lhe que tema utilizar:
<script>
  $(document).ready(function() {
    $(".tooltip").tooltipster(
        {
            theme: "tooltipster-noir";
        }
    );
  });
</script>
Isso permite-te ter total controlo do aspeto das tooltips... Se por exemplo tiveres várias fotos e quiseres aplicar estilos diferentes (e funcionalidades... mas falo disso mais à frente...), podes fazê-lo facilmente:
HTML:

<img class="tooltipNormal" src="img/sguimaraes.jpg" title="Sara Guimarães" alt="Sara Guimarães" />
<img class="tooltipLight" src="img/sguimaraes.jpg" title="Sara Guimarães" alt="Sara Guimarães" />
<img class="tooltipPunk" src="img/sguimaraes.jpg" title="Sara Guimarães" alt="Sara Guimarães" />
<img class="tooltipNoir" src="img/sguimaraes.jpg" title="Sara Guimarães" alt="Sara Guimarães" />
<img class="tooltipBorderless" src="img/sguimaraes.jpg" title="Sara Guimarães" alt="Sara Guimarães" />

JAVASCRIPT: 


<script>
  $(document).ready(function() {
    $(".tooltipNormal").tooltipster();
    $(".tooltipLight").tooltipster(
        {
            theme:"tooltipster-light"
        }
    );
    $(".tooltipPunk").tooltipster(
        {
            theme:"tooltipster-punk"
        }
    );
    $(".tooltipNoir").tooltipster(
        {
            theme:"tooltipster-noir"
        }
    );
    $(".tooltipBorderless").tooltipster(
        {
            theme:"tooltipster-borderless"
        }
    );
});
</script>

RESULTADO:

NOTA: Quando passei o theme à função tooltipster(), usei algo estranho com chavetas { ... }. Isto é um objeto JSON... Basicamente, permite-te definir um conjunto de chave -> valor. Nos casos acima, a chave é "theme", e o valor é o nome do tema que queres utilizar... Mas podes passar mais parâmetros a esta função.

{
    chave1:"valor associado à chave 1",
    chave2:"valor associado à chave 2",
    chave3:"valor associado à chave 3"
}
Estes vários parâmetros que consegue passar permite controlar ainda mais as tooltips. Todos os exemplos a seguir utilizam várias combinações destes parâmetros.
