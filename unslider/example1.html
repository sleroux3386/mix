Exemplo simples

Para usar o Unslider, precisas de adicionar 4 ficheiros ao teu HTML:

  • JQuery
  • Javascript do Unslider
  • CSSs do Unslider (a última versão tem dois CSS)

No caso do Weebox, o JQuery já vem carregado, e apenas precisas de adicionar os ficheiros do Unslider.

Numa página "limpa" (o Weebox é um caso especial), terias algo como: 

<head>
<link rel="stylesheet" type="text/css" href="css/unslider.css" />
<link rel="stylesheet" type="text/css" href="css/unslider-dots.css" />
<script type="text/javascript" src="js/jquery-2.2.4.min.js"></script>
<script type="text/javascript" src="js/unslider-min.js"></script>
</head>

Depois, o plugin é muito simples de utilizar:

  1. Crias uma div com uma classe qualquer.
  2. Dentro desta div, crias uma lista não-ordenada (<ul>...</ul>, lembras-te...), onde cada item desta lista é um dos slides (e sim, podes colocar texto, imagens, etc...).
  3. Executas o seguinte comando javascript: 
    $('.classQueDesteÀDiv').unslider();

Por exemplo:

HTML:


<div class="sara">
    <ul>
        <li>Sara</li>
        <li>Umbelina</li>
        <li>Macedo</li>
        <li>Guimarães</li>
    </ul>
<div/>

JAVASCRIPT: 


<script>
  $(document).ready(function() {
    $(".sara").unslider();
});
</script>

RESULTADO:

  • Sara
  • Umbelina
  • Macedo
  • Guimarães

Sim... Parece MUITO foleiro... Mas apenas se criou uma lista (sem estilos nenhum), adicionaram-se 4 pedaços de texto (também sem estilos nenhum) e lançou-se o slider... Com algum CSS e fuchsia à mistura, podes ""melhorar"" (ou não... não tenho jeito para desenho...) um pouco a coisa:

  • Sara
  • Umbelina
  • Macedo
  • Guimarães

No lindo e espetacular design acima, apenas tive criar uma classe, que juntei à div que envolve a lista (apenas criei uma class nova pois estou a utilizar a mesma class (sara) no exemplo "feio" e no exemplo "fuchsia"): 

.saraPretty{
    height: 50px;
    text-align: center;
    line-height: 50px;
    background-color: FUCHSIA;
    color: WHITE;
    font-weight: bold;
}

É possível melhorar ainda mais os slides... Por exemplo, substituir aqueles Prev e Next por setas, apresentar uns pontos por baixo dos slides para indicar o que está selecionado, automatizar a o slide (para fazer algo parecido com os thumbnails do Museu dos Terceiros), etc... Estes casos são explicados nos exemplos à esquerda.
