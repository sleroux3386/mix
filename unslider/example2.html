Exemplos "avançados"

Tal como no plugin Tooltipster, é possível configurar este plugin através de um objeto JSON, que passas como parâmetro ao executar a função 

unslider()
.

Nos exemplos iniciais, será utilizado o mesmo HTML do Exemplo simples. Nos exemplos finais, serão utilizadas imagens.

Slide automático

O slide automático não precisa de ter os botões para ver o slide seguinte. A transição é automática e podes configurar a velocidade da animação e o tempo que cada slide fica visível.

<script>
$(document).ready(function() {
    $(".slide1").unslider({
        autoplay: true
    });
});
</script>

Slide automático infinito

No exemplo anterior, quando chega ao último elemento da lista, o "carrossel" volta para trás para recomeçar. Podes alterar este comportamento, adicionando o parâmetro "infinite" com valor "true": 

<script>
$(document).ready(function() {
    $(".slide2").unslider({
        autoplay: true,
        infinite: true
    });
});
</script>

Slide automático com controlo de velocidade da animação

Nos exemplos anteriores, a animação é muito rápida (por omissão, demora 750ms a mudar de slide. Podes alterar este valor, através do parâmetro "speed":

<script>
$(document).ready(function() {
    $(".slide3").unslider({
        autoplay: true,
        infinite: true,
        speed: 2000
    });
    $(".slide4").unslider({
        autoplay: true,
        infinite: true,
        speed: 100
    });
});
</script>

O parâmetro speed APENAS controla o tempo que demora a animação de um slide para o outro... Para controlar o tempo em que o slide fica ativo, é necessário alterar um outro parâmetro...

Slide automático com controlo de velocidade da animação e atraso

Para controlar o tempo que um slide fica visível, tens de definir o parãmetro "delay". Por omissao, este tem um valor de 3000ms.

<script>
$(document).ready(function() {
    $(".slide5").unslider({
        autoplay: true,
        infinite: true,
        speed: 2000,
        delay: 500
    });
    $(".slide6").unslider({
        autoplay: true,
        infinite: true,
        speed: 2000,
        delay: 5000
    });
});
<script>

Slide automático com animação vertical

Nos exemplos anteriores, o slide é horizontal. Podes alterar este comportamento, através do parâmetros "animation". Por omissão, este tem o valor "horizontal". Mas podes alterar para "vertical" se quiseres...

<script>
$(document).ready(function() {
    $(".slide8").unslider({
        autoplay: true,
        infinite: true,
        animation: "vertical"
    });
});
</script>

Slides com imagens e com muitas coisas extras

Nos exemplos anteriores, apenas usei texto na lista. Mas como disse, podes colocar lá imagens...

Nos exemplos a seguir, são apresentadas algumas funcionalidades extras do plugin Unslider (desde controlo do slide por teclas, definição dos botões de navegação, etc...). Apenas vou dar algunas exemplos, e para mais documentação, podes consultar o site do plugin.

Nota: a escolha das imagens usadas nos exemplos a seguir foi totalmente aleatória.

<div class="slide10">
    <ul>
        <li><img src="img/cd1.png" with=200 height=200 /></li>
        <li><img src="img/cd2.jpg" with=200 height=200 /></li>
        <li><img src="img/cd3.jpg" with=200 height=200 /></li>
        <li><img src="img/cd4.jpg" with=200 height=200 /></li>
        <li><img src="img/cd5.jpg" with=200 height=200 /></li>
    </ul>
</div>

<script>
$(document).ready(function() {
    $(".slide10").unslider({
        autoplay: false,
        infinite: true,
        arrows: {
            prev: '<a class="unslider-arrow prev"<Slide anterior</a<',
            next: '<a class="unslider-arrow next"<Slide seguinte</a<'
        }
        speed: 1000    
    });
});
</script>
